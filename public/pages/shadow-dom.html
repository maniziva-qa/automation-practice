👻 Shadow DOM
Web Components • Encapsulated DOM

What is Shadow DOM?

Shadow DOM creates an encapsulated DOM tree attached to an element, hidden from the regular document DOM. Standard document.querySelector() cannot reach inside it.

💡
Selenium: Use JavascriptExecutor to pierce shadow roots.
((JavascriptExecutor) driver).executeScript("return arguments[0].shadowRoot.querySelector('#shadow-input')", host);

Playwright: Supports shadow DOM natively via page.locator('shadow-login-form >> #shadow-input') or >> combinator.
Cypress: Use .shadow() command: cy.get('shadow-login-form').shadow().find('#shadow-input')
