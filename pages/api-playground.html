🔌 API Playground
REST • GET • POST • PUT • DELETE

Available API Endpoints

Base URL: /api
GET/api/users— List all users (supports ?search=, ?role=, ?status=, ?delay=ms)
GET/api/users/:id— Get user by ID
POST/api/users— Create a new user (body: name, email, role, age, department)
PUT/api/users/:id— Update a user
DELETE/api/users/:id— Delete a user
POST/api/auth/login— Login (body: username, password)
GET/api/delayed?ms=3000— Delayed response
GET/api/flaky— 50% random failure (retry scenario)
POST/api/echo— Echo request body
